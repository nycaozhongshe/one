## 技术栈

- vue2.0
- vue-cli
- vue-router
- vue-resource
- vue-lazyload
- better-scroll 

## 实现功能

- [x] 首页列表
- [x] 滑动加载
- [x] 用户信息（点击头像）
- [x] 主题内容
- [x] 详情列表
- [x] 点赞（取消点赞）功能
- [x] touch事件
- [x] 图片懒加载
- [x] 无限加载 

## 执行项目

> 安装依赖  
> npm install  
> 启动服务  
> npm run dev  
> 运行  
> localhost:8080
